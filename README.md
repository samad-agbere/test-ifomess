# TEST IFOMESS

## Installation
*npm install*
*npm run dev*

## Endpoints
*backend*

- POST : /api/voeux pour enregistrement
- GET : /api/voeux pour recuperer tous les voeux
- GET : /api/voeux/:id pour recuperer tous un voeu dont l'id est specifié

*front*
/voeux : afficher les voeux en front

### Optionnel
**node backend/dbSeed.js** pour créer la table et seeder de fake data
